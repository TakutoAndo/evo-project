--keyweightcal()関数--
まず、キーを打ってもらって各キーの入力時間を測定する。
入力時間に応じて、キー位置に重みをつける。（0〜29）
--ObjFunc()関数--

最終的に、持ち点（全文字数*30）からのcount分減点方式で評価値を決める。
評価値を返す。
