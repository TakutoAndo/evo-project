----------------　評価方法　-----------------------

ホームポジションから指が移動してしまった回数をカウント
（指の重みづけをしたいので、小指位置にあるキーを押す場合はさらにカウント+1）
※ただし、１つ前と同じ文字だった場合、指の動きはないためカウントしない。
最終的に（全文字数ーカウント数）を評価値とする。
適応度（高）：指の移動が少ない
適応度（低）：指の移動が多い
